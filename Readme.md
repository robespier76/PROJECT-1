# <center> Проект. Анализ резюме из HeadHunter </center>

* В проекте: 
  * Осуществляется преобразование данных, содержащихся в базе данных резюме.
  * Исследуются зависимости в данных с помощью графической библиотеки Plotly.
  * Осуществляется очистка данных от пропусков, дубликатов и выбросов.
